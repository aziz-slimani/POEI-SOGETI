* Afin d'exécute les scripts de tests, les différentes actions doivent être suivie:
* 1) Ouvrir Git 
* 2) Se déplacer vers le répertoire souhaité 
* 3) Lancer la commande suivante dans git: git clone https://github.com/aziz-slimani/POEI-SOGETI.git 
* 4) Ouvrir un IDE (IntelliJ, Eclipse etc...)
* 5) Ouvrir le dossier POEI-SOGETI à travers IDE
* 6) Naviguer vers le répertoire où l'étape (3) a été exécutée
* 7) Ouvrir le répertoire \src\test\java sur IDE: Les trois classes ci-dessous peuvent être visibles
*  TestUS1  
*  TestUS2
*  TestUS3
*  TestUS4 
* Afin lancer l'exécution des différents tests automatisés situé dans le répertoire : (exécuter TestUS1 puis TestUS2 puis TestUS3 puis TestUS4)
* 8) Cliquez sur TestUS1
* 9) Cliquez sur le triangle de couleur verte située à gauche de la classes
* 10) Répéter l'action pour chacune des trois classes restantes.
